    widget(array $view_attrs = array()) : null | string

    Example:

    if in the model the field is

    $this->name = ORM::CharField(['max_length'=>30]);

    the form widget output can be customized as follows:

     $attributes = array(
    'class' => 'form-control',
    'style' => 'color: #000;'
    'style' => 'width:50%'
    );

    echo field->widget($attributes);

    which will output:

    <input type="text" name="name" value="" id="name" maxlength="30"
    style="width:50%" class="form-control" />
